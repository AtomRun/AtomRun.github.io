leetcode.389.找不同

2020-12-18 #leetcode
根据题意可以知道该题目范围在s和t的范围在a-z，并且新数组比老数组多了一个字符，所以可以将这道题目的解法将s和t转换为数组，然后遍历s将每个字母转为ASCII放到res中。然后再遍历t将其中的字符转为ASCII直接相减，这样剩下的res就是要找到的数据。 class Solution { public char findTheDifference(String s, String t) { int res = 0; for (int c : t.toCharArray()) res += c; for (int c : s.toCharArray()) res -= c; return (char) res; } }
